System Pipeline
1
Data Collection8,592 hourly records — temperature, time, consumption. Synthetic data mimics real smart-meter patterns.
2
Feature EngineeringCyclical encodings (sin/cos) for hour, month, day. Lag features (1h, 24h, 168h). 24-hour rolling mean.
3
Train / Test Split80% training (chronological), 20% test. StandardScaler normalises all features before training.
4
Linear RegressionOrdinary Least Squares (OLS). Minimises sum of squared residuals. Produces interpretable coefficients.
5
EvaluationMAE, RMSE, R², MAPE on held-out test set. Residual analysis confirms model assumptions.
6
ForecastingIterative 7-day ahead prediction using lag-chaining and rolling updates from the trained model.
