#EnergiSense – Energy Demand Forecasting Dashboard
Overview

EnergiSense is a Machine Learning-based Energy Demand Forecasting Dashboard designed to analyze and predict energy consumption patterns. The project uses a Linear Regression model with feature engineering techniques such as lag features, rolling averages, and cyclical time encoding to improve prediction accuracy.

The dashboard provides interactive visualizations for demand analysis, forecast trends, and model performance. It includes features like 7-day energy forecasting, weekday vs weekend consumption comparison, and evaluation metrics including R² Score, MAE, RMSE, and MAPE.
